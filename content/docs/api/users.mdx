# Users

User account management

## Get current user

<MethodBadge method="GET" path="/users/me" />

Returns authenticated user profile

### Security

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer YOUR_API_KEY`

## Update current user

<MethodBadge method="PATCH" path="/users/me" />

Update authenticated user profile

### Security

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer YOUR_API_KEY`

### Request Body

**Example:**

```json
{
    "display_name": "string",
    "avatar_url": "https://example.com",
    "settings": {}
}
```

### Code Examples

<CodeTabs samples={[
  {
    language: "cURL",
    code: `curl -X PATCH "https://searchaf.antfly.io/api/v1/users/me" \\\\
    -H "Authorization: Bearer YOUR_API_KEY" \\\\
    -H "Content-Type: application/json" \\\\
    -d '{
    "display_name": "string",
    "avatar_url": "https://example.com",
    "settings": {}
}'`
  },
  {
    language: "TypeScript",
    code: `const response = await fetch('https://searchaf.antfly.io/api/v1/users/me', {
    method: 'PATCH',
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        "display_name": "string",
        "avatar_url": "https://example.com",
        "settings": {}
    })
});

const data = await response.json();`
  },
  {
    language: "JavaScript",
    code: `fetch('https://searchaf.antfly.io/api/v1/users/me', {
    method: 'PATCH',
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        "display_name": "string",
        "avatar_url": "https://example.com",
        "settings": {}
    })
})
    .then(response => response.json())
    .then(data => console.log(data));`
  },
  {
    language: "Python",
    code: `import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY'
}

response = requests.patch(
    'https://searchaf.antfly.io/api/v1/users/me',
    headers=headers,
    json={
        "display_name": "string",
        "avatar_url": "https://example.com",
        "settings": {}
    }
)

data = response.json()`
  },
  {
    language: "Go",
    code: `package main

import (
    "bytes"
    "encoding/json"
    "net/http"
)

func main() {
    body := []byte(\`{
    "display_name": "string",
    "avatar_url": "https://example.com",
    "settings": {}
}\`)
    req, _ := http.NewRequest("PATCH", "https://searchaf.antfly.io/api/v1/users/me", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()
}`
  }
]} />

### Responses

  {
    statusCode: "200",

    body: `{

    "avatar_url": "https://example.com/avatar.jpg",
    "created_at": "2025-10-02T15:30:00Z",
