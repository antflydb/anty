# Audit

Audit log access

## List audit logs

<MethodBadge method="GET" path="/organizations/{org_id}/audit-logs" />

List audit logs for organization (requires admin+ role)

### Security

Provide your bearer token in the Authorization header when making requests to protected resources.

Example: `Authorization: Bearer YOUR_API_KEY`

### Parameters

| Name | Type | Location | Required | Description |
|------|------|----------|----------|-------------|
| `org_id` | string | path | Yes | Organization ID |
| `offset` | integer | query | No | Number of items to skip |
| `limit` | integer | query | No | Maximum number of items to return |
| `entity_type` | string (user, organization, project, api_key, subscription) | query | No |  |
| `entity_id` | string | query | No |  |
| `action` | string (create, read, update, delete) | query | No |  |
