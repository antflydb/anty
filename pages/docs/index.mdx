# Welcome to SearchAF Documentation

SearchAF is a powerful product search and discovery platform built on top of AntflyDB, a distributed vector database designed for the AI era.

## What is SearchAF?

SearchAF delivers cutting-edge product search and discovery capabilities that combine:

- **AI-Powered Semantic Search**: Understand user intent, not just keywords
- **Vector Database Technology**: Built on AntflyDB's proven distributed architecture
- **Enterprise-Grade Security**: OAuth integration, RBAC, and comprehensive audit logging
- **Platform Integrations**: Native support for Shopify, WooCommerce, and custom platforms

## Quick Start

Get started with SearchAF in just a few minutes:

1. **Create an Account**: Sign up for a SearchAF account
2. **Create an Organization**: Set up your organization workspace
3. **Create a Project**: Configure your first search project
4. **Generate API Keys**: Get your authentication credentials
5. **Integrate**: Add SearchAF to your e-commerce platform

## Key Features

### Semantic Search
Leverage AI-powered vector search to deliver results that truly match user intent. Our advanced embedding models understand context and meaning, not just keyword matches.

### Lightning Fast Performance
Built on AntflyDB's distributed architecture, SearchAF delivers sub-millisecond query performance at scale with horizontal scalability.

### Enterprise Ready
Production-grade infrastructure with enterprise security, compliance features, and comprehensive monitoring.

## Platform Support

SearchAF integrates seamlessly with:
